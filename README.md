# Passenger-Counter-App-
A simple and interactive passenger counter web app built using HTML, CSS, and JavaScript. This project is perfect for beginners who want to understand the basics of DOM manipulation, event handling, and state management in JavaScript.

📌 Description

The Passenger Counter App allows users to increment and track the number of passengers entering a location (like a bus, train, or station). It also provides the ability to save previous counts, making it a great mini-project to practice core JavaScript concepts.

This app demonstrates:

Updating UI dynamically
Handling button click events
Maintaining and displaying state
Basic styling with CSS
